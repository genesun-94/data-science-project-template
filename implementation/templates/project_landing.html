Introduction to the Project

We're given two datasets, training and test, to run a LASSO linear regression model in order to predict house prices in Iowa given a set of features. We also had to clean the data to ensure there were no nulls within the datasets so that the linear regression algorithm could run properly. You can click directly here to view the outputs (the predicted sale price of a house).

Feature Engineering

After cleaning up the data and filling the missing values/nulls, we created several new features using the data we were given (a method called feature engineering) in order to create more data than what was provided to us.

Scaled/Encoded/Transformed Data

Following this, we then scaled and transformed the quantitative data and also encoded the categorical data by using dummy variables.

Train/Test Split

Normally, this should be done in the very beginning right after importing the data, but for the sake of the exercise, we did this split here. I used a 75/25 train/test split.
